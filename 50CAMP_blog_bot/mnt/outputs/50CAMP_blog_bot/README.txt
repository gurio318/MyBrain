========================================================
 50CAMP ブログ自動化ボット - セットアップガイド
========================================================

【ファイル構成】
  generate_and_post.py  ← メインスクリプト
  config.py             ← 設定ファイル（APIキー等を入力）
  keywords.csv          ← キーワードリスト（順番に使用）
  run.bat               ← ダブルクリックで1記事生成（下書き保存）
  run_and_publish.bat   ← 即時公開バージョン
  scheduler_setup.bat   ← 毎朝自動実行の登録
  post_log.txt          ← 投稿ログ（自動作成）

========================================================
 STEP 1: Pythonのインストール
========================================================

1. https://www.python.org/downloads/ を開く
2. 最新版をダウンロード・インストール
3. インストール時「Add Python to PATH」に必ずチェック

確認方法: コマンドプロンプトで「python --version」と入力


========================================================
 STEP 2: config.py に情報を入力
========================================================

config.py をメモ帳で開いて以下を設定：

---- Claude API キー ----
  CLAUDE_API_KEY = "sk-ant-xxxxx..."
  → https://console.anthropic.com/ でAPIキーを取得

---- WordPress設定 ----
  WP_URL      = "https://gurio.work"
  WP_USERNAME = "ユーザー名"
  WP_APP_PASS = "アプリケーションパスワード"

  ★ アプリケーションパスワードの取得方法 ★
  1. gurio.work の WordPress管理画面にログイン
  2. 「ユーザー」→「プロフィール」を開く
  3. 一番下の「アプリケーションパスワード」セクションへ
  4. アプリ名に「BlogBot」と入力し「新しいアプリケーションパスワードを追加」
  5. 表示されたパスワード（xxxx xxxx xxxx xxxx xxxx xxxx）をコピー
  6. config.py の WP_APP_PASS に貼り付け

---- 投稿設定 ----
  WP_STATUS = "draft"    ← 最初は下書きがおすすめ
  （確認できたら "publish" に変更すると即時公開になります）


========================================================
 STEP 3: keywords.csv を確認・編集
========================================================

keywords.csv をExcelやメモ帳で開いて確認。
列の意味：
  keyword  … 検索キーワード（必須）
  monetize … マネタイズ手法（スクールアフィリ、など）
  product  … 紹介商品名（任意）
  status   … 空白=未使用 / 済=使用済み

新しいキーワードは一番下に追加するだけでOK。


========================================================
 STEP 4: テスト実行
========================================================

1. run.bat をダブルクリック
2. 記事が生成されてWordPressに下書き保存される
3. gurio.work/wp-admin で「投稿」→「下書き」を確認
4. 内容がOKなら公開ボタンを押す

コマンドプロンプトから実行する場合：
  python generate_and_post.py           # 1記事・下書き
  python generate_and_post.py --publish # 1記事・即時公開
  python generate_and_post.py --count 3 # 3記事まとめて
  python generate_and_post.py --keyword "50代 副業 始め方"  # キーワード直接指定


========================================================
 STEP 5: 毎日自動実行の設定（スケジューラ）
========================================================

1. scheduler_setup.bat を右クリック→「管理者として実行」
2. 登録完了メッセージが出たらOK
3. 以降、毎朝10:00に自動で1記事生成・下書き保存される

タスクの確認：
  タスクスケジューラ（スタート→「タスクスケジューラ」で検索）
  →「タスクスケジューラライブラリ」→「50CAMP_Blog_Auto」

スケジュール解除：
  コマンドプロンプトで「schtasks /delete /tn 50CAMP_Blog_Auto /f」


========================================================
 よくある質問
========================================================

Q. 「anthropic」モジュールが見つからないと言われる
A. run.bat が自動でインストールしますが、手動の場合は
   コマンドプロンプトで「pip install anthropic」を実行

Q. WordPressへの投稿でエラーが出る
A. ・WP_APP_PASS のスペース（空白）もそのまま貼り付けてOK
   ・WP_URL の末尾にスラッシュ（/）が入っていないか確認
   ・WordPressのREST APIが有効か確認（プラグインで無効化されていることがある）

Q. アフィリエイトリンクはどうする？
A. 生成された記事に [AFFILIATE_LINK_HERE] という文字が入ります。
   WordPress の下書きで検索して、実際のアフィリエイトURLに置き換えてください。

Q. 記事の品質を上げたい
A. config.py の PERSONA（キャラ設定）を編集してください。
   実際のエピソードや口ぐせを追加すると一層リアルになります。
